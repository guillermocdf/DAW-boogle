¿Cómo jugar Boggle?

Objetivo:
El objetivo del juego es encontrar la mayor cantidad posible de palabras válidas usando las letras del tablero antes de que se acabe el tiempo.

Formar palabras:
Las palabras se forman seleccionando letras del tablero.
Cada letra debe estar adyacente a la anterior:
- horizontalmente
- verticalmente
- en diagonal
No se pueden saltar letras.
Una misma casilla no puede usarse más de una vez en la misma palabra.

Reglas de las palabras:
Deben tener al menos 3 letras.
Deben ser palabras válidas del idioma correspondiente.
No se permiten abreviaturas ni palabras inventadas.

Tiempo de juego:
Cada partida tiene una duración limitada: 1, 2 o 3 minutos.
Durante ese tiempo, intentá formar la mayor cantidad de palabras posible.

Puntuación:
La puntuación depende de la longitud de cada palabra encontrada.

Letras	Puntos
3–4	    1 punto
5	    2 puntos
6	    3 puntos
7	    5 puntos
8+  	11 puntos

Gana quien obtiene la mayor puntuación al finalizar el tiempo.
